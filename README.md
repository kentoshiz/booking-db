# Проект: Система онлайн-бронирования отеля (Hotel Booking System)
## Этап 1: Проектирование и наполнение БД (Архитектор / DBA)

### Как развернуть базу данных локально:
1. Создайте пустую базу данных в pgAdmin с именем `hotel_booking_db`.
2. Откройте Query Tool для созданной базы и выполните скрипт структуры таблиц из файла `structure.sql`.
3. Чтобы наполнить базу тестовыми данными (5 000 000 строк), выполните скрипт из файла `seed.sql`.

### Что сделано на первом этапе:
- Спроектирована ER-диаграмма в 3NF (10 таблиц + изолированный лог аудит-логов).
- Реализовано помесячное партиционирование (Range-секционирование) таблицы `bookings` по дате заезда (`check_in`).
- База успешно заполнена тестовыми 5 000 000 строк через функцию `generate_series`.
- Добавлены стандартные B-tree индексы, а также оптимизирующие частичный и составной индексы.
